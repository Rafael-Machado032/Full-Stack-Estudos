# Tipos de Containers

| Classe            | Max-width      |
|-------------------|---------------|
| `.container`      | Varia (breakpoint) |
| `.container-sm`   | 540px         |
| `.container-md`   | 720px         |
| `.container-lg`   | 960px         |
| `.container-xl`   | 1140px        |
| `.container-xxl`  | 1320px        |
| `.container-fluid`| 100% (sempre) |

> **Obs:** Valores podem variar conforme a versão do Bootstrap.
```

# Tipos de Tabelas no Bootstrap

| Classe             | Descrição                       |
|--------------------|---------------------------------|
| `.table`           | Tabela básica                   |
| `.table-striped`   | Linhas alternadas com cor       |
| `.table-bordered`  | Bordas em todas as células      |
| `.table-hover`     | Destaque ao passar o mouse      |
| `.table-sm`        | Tabela compacta                 |
| `.table-responsive`| Rolagem horizontal em telas pequenas |
```
