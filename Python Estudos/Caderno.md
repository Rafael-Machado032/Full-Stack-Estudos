## Instalação do Python

https://www.python.org/downloads/


1. Na tela inicial do instalador marcha "Install launcher for all users" e "Add Python to PATH" e click "Instal Now"
2. Na proxima tela clique "Disable path length limit"
3. Apos instalado para confirma abra o CMD e digita 
```bash
    py -3 --version 
```
Ele retorna a versão do python
4. No Visual Studio vai em "Extensões" e procure "Python" da Microsoft e instale e pronto ja pode comcar os projetos
